# Dashboard (Next/React) consumindo API

Interface de produto com tabela, filtros, formulários e integração com backend.

## Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Route Handlers (API mock)
- Zod para validação

## Funcionalidades
- DataTable com busca, filtro por status e ordenação
- Formulário com validação de inputs
- Integração com API mock usando `fetch`
- Estrutura pronta para deploy no Vercel

## Como executar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Observação
A rota `/api/products` funciona como backend mock para fins de portfólio.
